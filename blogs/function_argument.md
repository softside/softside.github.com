
学习python函数参数的传递
=================

function(a,b,c,d)
----
不限定参数个数，以逗号隔开。
调用时必须提供相应的参数，且位置正确

function(a=a,b=b,c=c)
---
各个参数提供默认值，
调用时可不提供参数，全部采用默认值，可单独提供某个或多个"="赋值，
此时形如function(4),4是传给a的，，，**这是我考量这个问题的原因 **

function(*arg1)
----
这个传递的就是一个名字为arg1的list了

function(**arg1)
----
dict,不多说

最后
----
以上这些方式可以组合在一起形成复杂多样的形式，定义的时候遵循以下规则：
+ arg
+ arg=a
+ *arg
+ **arg
